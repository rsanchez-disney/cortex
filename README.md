# Cortex

Architectural knowledge graph for codebase understanding.

## Installation

```bash
koda cortex setup
```

Or manually:

```bash
uv tool install git+https://github.com/rsanchez-disney/cortex.git
```

## Usage

```bash
# Extract architecture from a project
koda cortex extract

# Start MCP server
koda cortex serve

# Check status
koda cortex status
```

## Releases

Binary releases are available as `.tar.gz.enc` assets attached to GitHub Releases.
